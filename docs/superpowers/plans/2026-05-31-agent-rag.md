# AgentRAG Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the first strong-AgentRAG slice: an analysis agent that plans, calls document RAG and optional web search, synthesizes a unified answer, and exposes execution trace in the UI.

**Architecture:** Keep existing RAG and MCP modules as tools. Add a focused backend orchestrator that decides which tools to call, preserves legacy `/chat` fields, and returns `agentAnswer`, `agentMode`, and `agentTrace`. Update the React conversation renderer to show the agent answer first, then the tool trace and existing source panels.

**Tech Stack:** Node ESM, Express, existing LangChain/OpenAI wrapper, React 18, Ant Design, node:test.
